# Months 2-6: Machine Learning Coding

## 📚 Overview

This phase covers **core machine learning techniques** applied to clinical problems:
- **Supervised Learning** (Months 2-3): Classification, regression, churn prediction
- **Unsupervised Learning** (Month 4): Clustering, dimensionality reduction
- **Time Series & Recommenders** (Month 5): Temporal analysis, patient recommendations
- **Deep Learning** (Month 6): Neural networks, CV, NLP

---

## 🎯 Learning Objectives

1. Build and evaluate supervised learning models
2. Understand unsupervised learning techniques
3. Handle time-series data from clinical trials
4. Implement recommender systems for healthcare
5. Apply neural networks to complex problems
6. Work with medical imaging (CV) and clinical text (NLP)

---

## 📁 Sub-Modules

### 1. Supervised Learning (Months 2-3)
- `supervised/` folder
- Topics: Regression, Classification, Gradient Boosting, Hyperparameter Tuning

### 2. Unsupervised Learning & Time Series (Months 4-5)
- `unsupervised-time-series-recommenders/` folder
- Topics: Clustering, PCA, ARIMA, Prophet, Recommender Systems

### 3. Deep Learning, CV & NLP (Month 6)
- `neural-nets-cv-nlp/` folder
- Topics: Neural Networks, CNNs, RNNs, Transformers, NLP
