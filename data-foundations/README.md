# Month 1: Data Foundations

## 📚 Overview

This module covers the **fundamentals of data analysis** for clinical datasets. You'll learn:
- **Pandas**: Data manipulation and exploration
- **SQL**: Querying clinical databases
- **EDA**: Exploratory Data Analysis on healthcare data
- **Visualization**: Communicating insights effectively

---

## 🎯 Learning Objectives

1. Load, clean, and preprocess clinical datasets
2. Perform exploratory data analysis (EDA) on healthcare data
3. Write efficient SQL queries to extract insights
4. Create meaningful visualizations for healthcare KPIs
5. Understand data quality issues specific to healthcare

---

## 📋 Topics Covered

### 1. Pandas Fundamentals (Week 1)
- DataFrames and Series
- Reading/writing data (CSV, Excel, JSON)
- Data cleaning (handling missing values, duplicates)
- Groupby operations and aggregations
- Merging and joining datasets

### 2. SQL for Healthcare Data (Week 2)
- SELECT, WHERE, JOIN operations
- Aggregate functions (COUNT, SUM, AVG)
- Window functions for patient timelines
- Subqueries and CTEs
- Performance optimization

### 3. Exploratory Data Analysis (Week 3)
- Descriptive statistics
- Distribution analysis
- Correlation and covariance
- Outlier detection
- Missing data patterns

### 4. Data Visualization (Week 4)
- Line plots, scatter plots, histograms
- Box plots, violin plots
- Heatmaps and correlation matrices
- Dashboard basics (Plotly, Seaborn)
