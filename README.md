# AI/ML Clinical Journey

## 📚 Overview

A comprehensive 13-month learning roadmap designed to take you from **data fundamentals through production-grade MLOps and GenAI engineering** in the clinical/healthcare domain.

---

## 🎯 Learning Roadmap

### **Phase 1: Foundations (Months 1-6)**

#### Month 1: Data Foundations
- **Topics**: Pandas, SQL, EDA, Data Visualization
- **Location**: `data-foundations/`

#### Months 2-6: Machine Learning Coding
- **Supervised Learning**: Churn prediction, classification
- **Unsupervised Learning**: Clustering, time-series, recommenders
- **Deep Learning**: Neural networks, CV, NLP
- **Location**: `ml-coding/`

### **Phase 2: Production & GenAI (Months 7-13)**

#### MLOps (Months 7-8)
- Model versioning, experiment tracking
- Deployment on Databricks/Fabric

#### RAG Systems (Month 9)
- Clinical document Q&A systems

#### Agentic AI (Month 10)
- Multi-agent workflows for trial operations

#### LLMOps & Fine-tuning (Months 11-13)
- Fine-tuning models, security, compliance

**Location**: `ml-genai-engineering/`

---

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Manikanta2991/scaler-ai-ml-clinical-journey.git
   cd scaler-ai-ml-clinical-journey
   ```

2. **Set up your environment**:
   ```bash
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```

3. **Navigate to the month** you're working on and follow the README in that directory.
