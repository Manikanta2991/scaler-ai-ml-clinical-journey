# Months 7-13: ML & GenAI Engineering

## 📚 Overview

Transition from experimentation to **production-ready systems** using MLOps, RAG, and agentic AI. This is the **core focus** of your 13-month journey.

---

## 🎯 Learning Objectives

1. Build scalable ML pipelines (MLOps)
2. Implement Retrieval-Augmented Generation (RAG) systems
3. Design multi-agent AI workflows for complex tasks
4. Fine-tune and deploy LLMs securely
5. Monitor, govern, and maintain production ML systems

---

## 📁 Structure

### 1. MLOps (Months 7-8)
- Feature engineering & stores
- Model versioning & registry
- Experiment tracking
- Pipeline orchestration
- Deployment strategies
- Monitoring & alerting

**Location**: `mlops/`

### 2. RAG Systems (Month 9)
- Vector databases
- Embeddings
- Chunking strategies
- Retrieval techniques
- LLM integration

**Location**: `rag-systems/`

### 3. Agentic AI (Month 10)
- Agent architectures
- Tool/function calling
- Multi-agent orchestration
- Reasoning & planning

**Location**: `agentic-ai/`

### 4. LLMOps, Fine-tuning & Security (Months 11-13)
- Prompt engineering
- Model fine-tuning (LoRA, QLoRA)
- HIPAA compliance
- Data privacy
- Security & access control

**Location**: `llmops-finetuning-security/`
