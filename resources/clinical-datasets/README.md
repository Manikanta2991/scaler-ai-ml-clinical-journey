# Clinical Datasets

## Available Datasets

### Public Datasets
- **MIMIC-IV** (PhysioNet) - EHR data, ICU records
- **eICU** (Collaborative) - Multi-center ICU data
- **CheXpert** - Chest X-ray interpretations
- **HAM10000** - Skin lesion classification

### How to Access
1. Create account on PhysioNet
2. Agree to data use agreements
3. Download via provided links
4. Document in your notebooks

### Data Formats
- CSV
- Parquet
- DICOM (medical imaging)
- HL7 (healthcare standards)
