# Research Papers

## Organization

Papers organized by topic:
- `deep-learning/` - Foundational DL papers
- `nlp/` - NLP research
- `medical-imaging/` - Computer vision for healthcare
- `clinical-ai/` - AI applications in clinical settings
- `fairness/` - Bias & fairness in AI

## Key Papers to Read

1. Attention is All You Need (Vaswani et al., 2017)
2. BERT: Pre-training (Devlin et al., 2018)
3. Transformer-based Clinical NLP (Alsentzer et al., 2019)
4. Fairness in Healthcare ML (Obermeyer et al., 2019)
5. Federated Learning for Healthcare (Yang et al., 2019)

## How to Use
1. Read papers in `papers/` folder
2. Implement findings in notebooks
3. Cite in your projects
4. Share implementations in `implementations/`
